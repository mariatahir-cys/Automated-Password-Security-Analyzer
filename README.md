# Password Security Assessment & Risk Analysis Tool

## Description
This project is a Password Security Analyzer developed in Assembly Language using Irvine32 library. It checks how strong a password is based on character types, length, and security rules.

## Features
- Checks uppercase letters
- Checks lowercase letters
- Checks digits
- Checks special characters
- Classifies password as Weak, Medium, or Strong
- Shows reasons for weak passwords
- Displays brute-force risk level

## How it works
User enters a password, and the program:
1. Scans each character
2. Checks security conditions
3. Calculates strength score
4. Displays result with reasons

## Tools Used
- Assembly Language (x86)
- Irvine32 Library
- Visual Studio

## Output
- Weak → Easy to crack
- Medium → Moderate security
- Strong → Very hard to crack
